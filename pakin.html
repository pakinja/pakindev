Pakin Ciencia de Datos

Pakin Ciencia de Datos es un repositorio de conocimiento 100% abierto en el que se publican análisis, desarrollos, tutoriales y cursos de ciencia de datos que giran alrededor de problemas y fenómenos de impacto y que son elaborados en lenguajes de programación R, Python, HTML, SQL, Shell, entre otros.


Este es un sitio no sólo para quienes quieren aprender ciencia de datos, sino también para aquellos que buscan información de calidad que puede ser aplicada para generar cambios positivos en las sociedades y ecosistemas.

Misión: Contribuir a democratizar la ciencia de datos en México y América Latina.


Visión: Ser el nodo de ciencia de datos más impactante de habla hispana.


Convicción: Generar valor para las sociedades y ecosistemas formando redes complejas.
